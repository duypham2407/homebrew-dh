# homebrew-dh

Homebrew tap for [dh](https://github.com/duypham2407/dh-kit) — local-first AI coding assistant for macOS and Linux.

## Install

```sh
brew tap duypham2407/dh
brew install dh
```

Or in one line:

```sh
brew install duypham2407/dh/dh
```

## Upgrade

```sh
brew upgrade dh
```

## Uninstall

```sh
brew uninstall dh
brew untap duypham2407/dh
```
